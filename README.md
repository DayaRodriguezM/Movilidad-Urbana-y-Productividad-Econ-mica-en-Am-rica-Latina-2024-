# Movilidad-Urbana-y-Productividad-Econ-mica-en-Am-rica-Latina-2024-
Este proyecto del bootcamp analiza de la relación entre congestión urbana y productividad económica en 15 ciudades de América Latina usando Python, pandas y visualizaciones interactivas. Incluye datos de TomTom Traffic Index y OECD.
